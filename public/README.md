# Public Assets

This folder contains static assets that are served directly by Next.js.

## Logo

Please place your logo file here as `logo.png`. The logo will be automatically used in:
- Header navigation
- Footer
- Other branding locations

### Supported Formats
- PNG (recommended)
- SVG (for scalable logos)
- JPG/JPEG

### Recommended Dimensions
- Header logo: 40px height (width will scale proportionally)
- Footer logo: 40px height (width will scale proportionally)

If you have a different logo file name, please update the references in:
- `components/layout/Header.tsx`
- `components/layout/Footer.tsx`
